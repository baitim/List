<h1 align="center">List</h1>

## Description

 Implementation of doubly linked list.

## How to use

1. Clone <br>
        write <code>git clone https://github.com/baitim/List.git</code> in terminal

2. Go to folder <br>
        write <code>cd List</code> in terminal

3. Build & run <br>
        write <code>make</code> in terminal
